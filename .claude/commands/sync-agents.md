# Sync Agents

Sync the team's agents, skills, commands, and hooks from this repo to `~/.claude/` by running the sync script.

## Steps

1. Run the sync script:

```bash
bash "$CLAUDE_PROJECT_DIR/.claude/commands/scripts/sync-agents.sh"
```

2. Report the output to the user.

## Usage

```
/sync-agents
```

Run from anywhere inside this repo after pulling updates.
